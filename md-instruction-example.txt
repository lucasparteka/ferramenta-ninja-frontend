# CPF Generator & Validator – Technical & SEO Requirements

## Overview

This document defines the implementation of the **CPF Generator & Validator Tool**.

The tool allows users to:

* Generate valid CPF numbers
* Validate existing CPF numbers
* Optionally format CPF (###.###.###-##)

This is a **high-demand tool in Brazil**, widely used by developers, testers, and users filling forms.

# Logic Requirements

## CPF Structure

CPF = 11 digits:

* First 9 digits: base
* Last 2 digits: verification digits

---

## Generator Logic

### Steps

1. Generate 9 random digits
2. Calculate first verification digit
3. Calculate second verification digit
4. Combine into full CPF

---

## Validation Logic

### Rules

* Must have 11 digits
* Reject sequences like:

  * 00000000000
  * 11111111111
* Validate verification digits

---

## Core Functions

```ts id="r1f8kx"
function generateCPF(formatted: boolean): string

function generateMultipleCPF(count: number, formatted: boolean): string[]

function validateCPF(cpf: string): boolean

function formatCPF(cpf: string): string
```

---

## Verification Digit Algorithm

### First Digit

* Multiply digits by weights (10 → 2)
* Sum results
* Calculate remainder:

  * (sum * 10) % 11
* If result = 10 → use 0

---

### Second Digit

* Multiply digits (including first digit) by weights (11 → 2)
* Same remainder rule

---

## Randomness

* `Math.random` is acceptable for generation
